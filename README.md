# Simulador de Sinapsis Celulares
**Autor:** Antonio Gabucio López  
**Fecha:** Marzo 2026  
**Versión:** 1.0  

### Descripción:
Esta herramienta es un simulador fisiológico de sinapsis celulares. Asemeja las variaciones de potenciales de acción en procesos sinápticos donde podemos elegir la naturaleza y el estímulo de células presinápticas y simular su efecto en células post-sinápticas.

### Herramientas de Software (Open Source):
- Interfaz: Flet
- Procesamiento: NumPy y Matplotlib

### Agradecimientos / Notas:
Basado en: BASIC PRINCIPLES OF SYNAPTIC PHYSIOLOGY ILLUSTRATED BY A COMPUTER MODEL. Michael J. Davis. Advances in Physiology Education 2001 25:1, 1-12 
Código estructurado y depurado con la asistencia de IA (Google Gemini).
